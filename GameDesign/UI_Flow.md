# 📊 UI 흐름도 (UI Flowchart)

> 게임의 주요 UI 화면과 화면 간의 전환 흐름을 정리합니다.

---
## 1. 메인 화면 (Main Menu UI)

**[화면 구성]**
- 게임 타이틀
- '게임 시작' 버튼
- '스킨 변경' 버튼
- '설정' 버튼
- '게임 종료' 버튼

**[흐름]**
- `(시작)` -> **메인 화면** 표시
- `'게임 시작' 버튼 클릭` -> **로딩 UI** 표시 및 매치메이킹 시작
- `'스킨 변경' 버튼 클릭` -> **스킨 선택 화면**으로 이동
- `'설정' 버튼 클릭` -> **설정 팝업** 표시 (보이스 챗 토글 설정 등)
- `'게임 종료' 버튼 클릭` -> 게임 애플리케이션 종료

---
## 1.5. 스킨 선택 화면 (Skin Select UI)

**[화면 구성]**
- **스킨 미리보기**: 화면 중앙에 현재 선택된 스킨의 3D 모델 표시.
- **이동 버튼**:
    - 좌/우 화살표 아이콘 (마우스 클릭)
    - 키보드 `A`, `D` 또는 `Left Arrow`, `Right Arrow` 키 지원.
- **'장착' 버튼**: 선택한 스킨 하단에 위치. 현재 장착 중인 스킨은 '장착됨'으로 표시.
- **'닫기' 버튼**: 화면 우측 상단 `X` 버튼 또는 `ESC` 키.

**[흐름]**
- `(진입)` -> **스킨 선택 화면** 표시 (기본 스킨 또는 현재 장착 중인 스킨 포커스)
- `좌/우 조작` -> 스킨 모델 교체 (애니메이션 포함 추천)
- `'장착' 버튼 클릭` -> 해당 스킨을 플레이어 데이터에 저장
- `'닫기' 또는 ESC` -> **메인 화면**으로 복귀

## 2. 인게임 화면 (In-game UI)

**[화면 구성]**
- **게임 타이머**: 화면 상단 중앙 (예: 150초부터 카운트다운)
- **미션 알림**: 화면 중앙 (예: "단체 댄스 시작!")
- **댄스 액션 안내 UI**:
    - '가면 댄스' 또는 '단체 댄스' 시, 화면에 1, 2, 3, 4번 키에 해당하는 댄스 아이콘(사람 모양 그림)이 표시됩니다.
    - 각 아이콘은 해당하는 댄스 애니메이션을 작게 보여줍니다.
- **단체 댄스 이벤트 연출**:
    - **1. 하늘 암전 및 천막 하강**: 하늘이 검게 변하고, 맵 가장자리를 따라 천막이 내려오며 주변이 어두워집니다.
    - **2. 미러볼 하강**: 중앙 공중에서 미러볼이 내려옵니다.
    - **3. 조명 및 음악 시작**: 미러볼 조명이 켜지며 단체 댄스 BGM이 시작됩니다.
- **술래 전용 UI (Seeker Only)**:
    - **조준선**: 화면 중앙에 스턴건 발사를 위한 조준선이 표시됩니다.
    - **스킬 쿨타임**: 화면 우측 하단 ('E' 키 아이콘과 쿨타임 표시)
- **다음 가면 댄스 예고**: (삭제됨 - 술래가 직접 스킬로 발동)
- **보이스 상태**: 화면 좌측 하단 (마이크 아이콘. OFF/ON/말하는 중 표시)
- **조작 키 안내**: 화면 하단 (필요 시)

**[흐름]**
- `매치메이킹 성공` -> **인게임 화면** 표시
- `게임 시간 종료` OR `모든 플레이어 탈락` -> **스테이지 종료 화면** 표시

---
## 3. 스테이지 종료 화면 (Stage End UI)

**[화면 구성]**
- **결과 텍스트**: "VICTORY" 또는 "DEFEAT"
- **통계 정보**:
    - 평균 반응 속도
    - 나의 가면 기록
- **'메인으로' 버튼**

**[흐름]**
- `게임 종료` -> **스테이지 종료 화면** 표시
- `'메인으로' 버튼 클릭` -> **메인 화면**으로 전환

---
## 전체 흐름 요약

`앱 실행` -> **[메인 화면]** (<-> **[스킨 선택]**) -> `'게임 시작'` -> `(로딩 UI)` -> **[인게임 화면]** -> `게임 종료` -> **[스테이지 종료 화면]** -> `'메인으로'` -> **[메인 화면]**
