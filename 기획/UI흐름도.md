# 📊 UI 흐름도 (UI Flowchart)

> 게임의 주요 UI 화면과 화면 간의 전환 흐름을 정리합니다.

---
## 1. 메인 화면 (Main Menu UI)

**[화면 구성]**
- 게임 타이틀
- '게임 시작' 버튼
- '게임 종료' 버튼

**[흐름]**
- `(시작)` -> **메인 화면** 표시
- `'게임 시작' 버튼 클릭` -> **로딩 UI** 표시 및 매치메이킹 시작
- `'게임 종료' 버튼 클릭` -> 게임 애플리케이션 종료

---
## 2. 인게임 화면 (In-game UI)

**[화면 구성]**
- **게임 타이머**: 화면 상단 중앙 (예: 150초부터 카운트다운)
- **미션 알림**: 화면 중앙 (예: "단체 댄스 시작!", "빨간 가면 댄스!")
- **댄스 액션 안내 UI**:
    - '가면 댄스' 또는 '단체 댄스' 시, 화면에 1, 2, 3, 4번 키에 해당하는 댄스 아이콘(사람 모양 그림)이 표시됩니다.
    - 각 아이콘은 해당하는 댄스 애니메이션을 작게 보여줍니다.
- **단체 댄스 이벤트 연출**:
    - 이벤트 시작 시, **공중에 미러볼이 나타나고 조명이 반짝이는** 등의 시각적 연출이 추가됩니다.
- **술래 전용 UI (Seeker Only)**:
    - **다음 가면 댄스 예고**: 다음에 어떤 색상의 가면 댄스가 시작될지 미리 알려주는 텍스트 또는 아이콘 (예: "Next: Red Mask Dance").
- **조작 키 안내**: 화면 하단 (필요 시)

**[흐름]**
- `매치메이킹 성공` -> **인게임 화면** 표시
- `게임 시간 종료` OR `모든 플레이어 탈락` -> **스테이지 종료 화면** 표시

---
## 3. 스테이지 종료 화면 (Stage End UI)

**[화면 구성]**
- **결과 텍스트**: "VICTORY" 또는 "DEFEAT"
- **통계 정보**:
    - 평균 반응 속도
    - 나의 가면 기록
- **'메인으로' 버튼**

**[흐름]**
- `게임 종료` -> **스테이지 종료 화면** 표시
- `'메인으로' 버튼 클릭` -> **메인 화면**으로 전환

---
## 전체 흐름 요약

`앱 실행` -> **[메인 화면]** -> `'게임 시작'` -> `(로딩 UI)` -> **[인게임 화면]** -> `게임 종료` -> **[스테이지 종료 화면]** -> `'메인으로'` -> **[메인 화면]**
